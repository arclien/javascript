# 챕터1

```
index.html

<!doctype html>
<html>
  <head>
    <link rel="stylesheet" href="main.css">
  </head>
  <body>
    <h1>hello world</h1>
    <script src="main.js"></script>
  </body>
</html>
```
- head에 있는 css파일과, body 마지막에 있는 script import를 이해하자.
  - 웹 퍼포먼스에 약간의 영향을 준다.

- 비동기적 이벤트
  - 이벤트가 언제 일어날지 알 수 없는 이벤트
  - 객체에 이벤트 핸들러를 연결 해서 이벤트 처리
